    RegisterSingleton(ITargetContainer, Type, Type, IMemberBindingBehaviour)

    Registers the type objectType as a singleton (SingletonTarget) in the target container using either objectType as the service type, or serviceType instead - if it's provided.

    The instance will be built automatically with constructor injection (and, optionally, property injection if a propertyBindingBehaviour is passed) by leveraging the ConstructorTarget or

    GenericConstructorTarget targets.
