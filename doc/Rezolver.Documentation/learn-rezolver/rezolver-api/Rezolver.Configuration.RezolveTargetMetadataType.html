    MetadataList

    Describes a list of IRezolveTargetMetadata instances - to be used for metadata objects that require multiple metadata objects.

    Can be created to feed a multiple instance registration for a single type (e.g. registering multiple instances of IFoo to a container, so that you can resolve all of them by resolving IEnumerable), or simply as a collection of targets that are to be used to build an array that is to be passed as a constructor argument or property.
