    Class ConstructorTarget

    The most common target used for building new objects. Represents binding to a type's constructor with zero or more arguments supplied by other ITargets.

    In addition to using the Rezolver.ConstructorTarget. constructor you can also use the factory methods - such as Auto(IPropertyBindingBehaviour).
