    Parameters
    Name Type Description
    entry IConfigurationEntry

    The entry to be transformed.
    context ConfigurationAdapterContext

    The context for the operation.
    Returns
    Type Description
    RezolverBuilderInstruction

    If the entry can be converted into a RezolverBuilderInstruction, then an instance of that type, otherwise null.
    Remarks

    The default behaviour is to:

    1. Attempt to convert the entry to an ITypeRegistrationEntry
    2. Parsing its type references in Types
    3. Constructing an ITarget from the entry's TargetMetadata through a call to its CreateRezolveTarget(Type[], ConfigurationAdapterContext, IConfigurationEntry) method.
    4. If that returns a non-null target, then a RegisterInstruction is created and returned.
