    Interface IRezolveTargetEntry

    Records one or more IRezolveTargets that have been registered against a given type.

    Also implements the IRezolveTarget interface so that you can request targets for the different types that are ultimately supported by the implementation based on the targets that have been added to it.

    Attach(ITargetContainer, IRezolveTargetEntry)

    ONLY called by rezolver builder classes, when an entry is attached to a builder by being passed directly to Register(ITarget, Type), optionally replacing an existing entry registered for the same type.

    Note that after this has been called once, the entry is considered to be attached to the builder that the replacing entry was attached to, and the method cannot be called again.
