Class RezolveContext

Context of a call to an IRezolver's Resolve method. The rezolver is included in the context to allow IRezolveTarget-generated code to refer back to the rezolver.

This also allows us to retarget compiled targets at other rezolvers (e.g. child rezolvers that override existing registrations or define new ones).
