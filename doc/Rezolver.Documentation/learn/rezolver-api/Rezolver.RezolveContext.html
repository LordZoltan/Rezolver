    Class RezolveContext

    Context of a call to an IRezolver's Resolve method. The container is included in the context to allow code generated from Rezolver.ITarget.CreateExpression(Rezolver.CompileContext) to refer back to the container.

    This also allows us to retarget compiled targets at other containers (e.g. OverridingContainers that override existing registrations or define new ones).
