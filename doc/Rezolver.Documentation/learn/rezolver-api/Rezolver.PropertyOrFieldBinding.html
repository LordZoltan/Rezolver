    DeriveAutoPropertyBinding(Type, Boolean)

    Static factory method that creates bindings for all publicly writable instance properties (and, optionally, fields) of the given type. Each property/field is bound to a RezolvedTarget instance - meaning that, at runtime, values for those properties or fields will be resolved from the container by type.
