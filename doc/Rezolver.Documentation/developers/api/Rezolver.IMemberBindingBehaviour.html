    Interface IMemberBindingBehaviour

    Describes a type which discovers and creates property/field bindings, typically for use when creating a new instance.

    Namespace:Rezolver
    Assembly:rezolver-docfx.dll
    Syntax
    public interface IMemberBindingBehaviour
    Remarks

    The default implementation of this is DefaultMemberBindingBehaviour, which always binds all publicly writable instance properties and publicly accessible instance fields to auto-generated ResolvedTarget targets.
