    Syntax
    public class TargetExpression : Expression
    Remarks

    The ExpressionTargetBuilder uses this class extensively when translating expressions into targets and back again. It's highly unlikely you'll need to use it in your code.

    It acts as a placeholder for targets until the point at which the expression compiler wants to build a complete expression tree for a target.
