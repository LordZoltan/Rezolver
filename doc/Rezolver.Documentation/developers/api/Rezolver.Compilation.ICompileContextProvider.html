    Interface ICompileContextProvider

    Interface for an object which creates ICompileContext instances for when the system needs to compile an ITarget into an ICompiledTarget.

    Namespace:Rezolver.Compilation
    Assembly:Rezolver.dll
    Syntax
    public interface ICompileContextProvider
    Remarks

    In normal operation, this interface is closely related to the ITargetCompiler interface because the core ContainerBase class (which provides most of the default implementation for IContainer) obtains a new ICompileContext by resolving an instance of this interface, and calling the CreateContext(ResolveContext, ITargetContainer, IContainer) method, passing the result to the CompileTarget(ITarget, ICompileContext) method, along with the target to be compiled.

    Frequently, implementations of ITargetCompiler will also implement this interface to ensure that the compilation context contains everything the compiler needs to do its work.
