    Class DecoratingTargetContainer

    Implements service decoration in an ITargetContainer, producing instances of the DecoratorTarget when Fetch(Type) or FetchAll(Type) are called.

    The best way to add a decorator to your target container is to use the extension method RegisterDecorator<TDecorator, TDecorated>(ITargetContainer) or its non-generic equivalent.

    Remarks

    This class does not implement ITarget, rather it's an ITargetContainer into which other targets can be added, and when Fetch(Type) or FetchAll(Type) are called, a temporary DecoratorTarget is created which wraps around the targets that have been registered within and which will ultimately create instances of DecoratorType

    Remarks

    The decorator target does not accept registrations directly; rather it passes the call on to its inner container which could be a TargetListContainer, or GenericTargetContainer in the most basic cases; or it could be another DecoratingTargetContainer in situations where a type has had multiple decorators registered against it.
