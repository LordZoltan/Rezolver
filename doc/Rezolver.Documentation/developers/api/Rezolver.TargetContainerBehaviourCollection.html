    Class TargetContainerBehaviourCollection

    An ITargetContainerBehaviour which contains zero or or more other ITargetContainerBehaviour objects. Behaviours can depend on other behaviours, and this collection ensures that they are applied in the correct order.

    Remarks

    The implementation runs through each behaviour that has been added to the collection, in dependency order, calling its Attach(ITargetContainer) method, passing the targets to each.
