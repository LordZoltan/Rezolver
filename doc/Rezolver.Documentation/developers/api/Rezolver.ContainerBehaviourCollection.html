    Class ContainerBehaviourCollection

    An IContainerBehaviour which contains zero or or more other IContainerBehaviour objects. Behaviours can depend on other behaviours, and this collection ensures that they are applied in the correct order.

    Remarks

    The implementation runs through each behaviour that has been added to the collection, in dependency order, calling its Attach(IContainer, ITargetContainer) method.
