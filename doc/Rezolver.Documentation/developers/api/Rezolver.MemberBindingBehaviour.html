    Class MemberBindingBehaviour

    Static accessor for the common member binding behaviours provided by Rezolver, also provides an entry point to build custom binding behaviours using the fluent API exposed by IMemberBindingBehaviourBuilder<TInstance> and MemberBindingBuilder<TInstance, TMember> - via the For<TInstance>() method.

    Remarks

    The ConstructorTarget and GenericConstructorTarget classes can be provided with an IMemberBindingBehaviour when created. If one is not set, then instead they will attempt to get their behaviour via the options API (see OptionsTargetContainerExtensions) from the ITargetContainer in which they have been registered.
