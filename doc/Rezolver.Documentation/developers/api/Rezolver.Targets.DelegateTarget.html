    Syntax
    public class DelegateTarget : TargetBase, ITarget
    Remarks

    The delegate must be non-void and can have any number of parameters.

    A compiler must ensure that any parameters for the Factory are automatically resolved from the container, and that a parameter of the type ResolveContext will receive the context passed to the Resolve(ResolveContext) method call for which this target is being compiled and/or executed.

    Remarks

    Whilst this can be easily obtained from the delegate yourself (by using the GetMethodInfo(Delegate) extension method) however, this class also uses it to determine the DeclaredType of the target or whether the delegate is actually compatible with the one supplied on construction, therefore if you need to introspect the delegate, you might as well use this.
