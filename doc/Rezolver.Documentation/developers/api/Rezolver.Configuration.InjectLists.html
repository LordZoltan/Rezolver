    Class InjectLists

    This configuration will enable automatic injection of List<T>, IList<T> and IReadOnlyList<T> when applied to an ITargetContainer, so long as there aren't already registrations for those types.

    Remarks

    If this configuration is added to a CombinedTargetContainerConfig, it can be disabled by adding another configuration to set the EnableListInjection option to false - typically via the ConfigureOption<TOption>(CombinedTargetContainerConfig, TOption) method (note - the order that the configs are added doesn't matter).

    The underlying behaviour relies on registrations of IEnumerable<T> to be present when the constructor for the list type is bound, as it expects to bind to the List<T>(IEnumerable<T>) constructor.

    The easiest way to achieve this is also to ensure that the InjectEnumerables configuration is enabled (which it is, by default) - which guarantees that any IEnumerable<T> can be resolved - even if empty.
