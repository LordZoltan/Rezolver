    Interface IChildTargetContainer

    An ITargetContainer that inherits all registrations from a (Parent) target container. If it cannot resolve a target for a particular type, will defer to its parent for fallback.

    Remarks

    Note that the framework does not require that enumerables of targets (retrieved by calling are merged between a child and parent container.

    Typically, as soon as one registration exists in a child container for the same type as in the parent, it overrides all registrations in the parent for that same type.
