    Class Functions

    This type is only used when using expressions as targets (via the ExpressionTarget type) - it's functions server no actual purpose other than to act as hooks to create specific ITarget objects in place of static code.

    For example, the Resolve<T>() function is used to trigger the creation of a ResolvedTarget in its place - thus allowing expressions to leverage the full power of the Rezolver API all through a simple method call.
