    Remarks

    Resolving an object via a scope does not guarantee that it will be tracked. Ultimately, it's up to the behaviour of the individual underlying targets to determine whether they should interact with the scope.

    Indeed, all this extension method does is to forward the method call on to the Container of the given scope, ensuring that the scope is set on the ResolveContext that is passed to its Resolve(ResolveContext) method.
