    RegisterAutoFactory(IRootTargetContainer, Type)

    Registers an AutoFactoryTarget for the given delegateType, enabling the container to automatically build and inject a delegate which uses the container to produce a result.

    This method and its overload can be used for any delegate type.

    Declaration
    public static void RegisterAutoFactory(this IRootTargetContainer targets, Type delegateType)
    Parameters
    Type Name Description
    IRootTargetContainer targets

    Required. The target container in which the registration is to be made.
    Type delegateType

    Required. Must have a non-void return type and all its parameters must be of distinct types.

    RegisterAutoFunc<TResult>(IRootTargetContainer)

    Enables the automatic injection of a Func<TResult> for the given TResult

    Declaration
    public static void RegisterAutoFunc<TResult>(this IRootTargetContainer targets)
    Parameters
    Type Name Description
    IRootTargetContainer targets

    Required. The IRootTargetContainer into which the newly created target will be registered
    Type Parameters
    Name Description
    TResult

    The return type of the delegate - equivalent to the service type that is to be resolved from the container when the delegate is called.
    Remarks

    Note that scoping is honoured for the delegate call; with an injected auto-factory being bound to the scope from which it is resolved.

    This methods ensures that enumerables of the delegate type are also injectable.
