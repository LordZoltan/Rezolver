    Interface IContainerConfig

    A configuration callback for instances of IContainer (which also use ITargetContainer as the source of their registrations).

    Namespace: Rezolver
    Assembly: Rezolver.dll
    Syntax
    public interface IContainerConfig
    Remarks

    While similar to ITargetContainerConfig, this is specifically used for new container instances - since some configuration (setting of options etc) only applies to IContainer and not ITargetContainer.

    A configuration instance can be passed to any of the provided Container-derived types on construction (see the Container(IRootTargetContainer, IContainerConfig) constructor). If one is not passed then the DefaultConfig is used.

    The built-in config implementations register services or set options in the target container passed to Configure(IContainer, IRootTargetContainer).
