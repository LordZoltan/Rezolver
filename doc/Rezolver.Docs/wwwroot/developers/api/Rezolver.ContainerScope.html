    Root

    Root scope to be used for 'top-level' object tracking.

    Note - when a Container is used, this should be the first scope created via the CreateScope() method.

    With ScopedContainer it will be the scope that lives inside that container.

    So, as a result of this, this property could point to a scope that's not actually a parent of Parent
