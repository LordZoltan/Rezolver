    Remarks

    This target is produced automatically by functionality such as FetchAllCompatibleTargets(IRootTargetContainer, Type) for any target whose registered type is not exactly the same as the type requested. The target's identity (chiefly determined by its Id) is always equal to the Target that it wraps, thus allowing it to masquerade as the wrapped target.
