    CreateContext(ResolveContext, ITargetContainer)

    Creates a compilation context for the given resolveContext - which is used to determine the RequestedType that the factory which is eventually compiled should return (through either the CompileTarget(ITarget, ICompileContext) or CompileTarget<TService>(ITarget, ICompileContext) methods).
