    Remarks

    The implementation registers a special internal target type which implements ICompiledTarget simply by returning the context passed to its GetObject(IResolveContext) method.

    This configuration is applied to the DefaultConfig automatically, and cannot be disabled through the use of options. Either it's in the configuration, or its not.
