    CreateTargetContainerForServiceType(IRootTargetContainer, Type)

    Calls GetContainerRegistrationType(Type) to get the correct registration type for the target container that should own a target that will be registered against the given serviceType. Once obtained, that type is then passed to the CreateTargetContainer(Type) to create the correct ITargetContainer for that container registration type.

    This is typically used by 'advanced' target containers which wrap others under potentially any type and which therefore need to reuse the root container's logic for ensuring that registrations are handled correctly.

    FetchAllCompatibleTargets(IRootTargetContainer, Type)

    Fetches all targets from the rootContainer registered against service types which are either equal to or compatible with the given serviceType.

    Declaration
    public static IEnumerable<ITarget> FetchAllCompatibleTargets(this IRootTargetContainer rootContainer, Type serviceType)
    Parameters
    Type Name Description
    IRootTargetContainer rootContainer

    The root target container whose targets are to be searched
    Type serviceType

    The service type for which targets are sought.
    Returns
    Type Description
    IEnumerable<ITarget>

    An enumerable of targets which match the serviceType. If no targets match, then the enumerable will be empty.
    Remarks

    This is effectively a covariant version of the FetchAll(Type) function, and is used by built-in functionality such as automatic enumerable injection (when EnableEnumerableCovariance is true) and automatic factory injection Func<TResult>.

    Note: the targets are de-duped before being returned, so if one target is registered multiple times against different compatible service types, it will still only appear once in the output enumerable.
