    BuildResolveLambda(IExpressionCompiler, ITarget, IExpressionCompileContext)

    This method is a shortcut for building a lambda expression directly from an ITarget. It calls Build(ITarget, IExpressionCompileContext) and passes the result to BuildResolveLambda(Expression, IExpressionCompileContext), which should yield an optimised lambda expression for the expression produced from the target which can then be compiled and used as the factory for that target.
