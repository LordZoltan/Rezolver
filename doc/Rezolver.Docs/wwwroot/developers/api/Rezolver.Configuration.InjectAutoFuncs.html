    Class InjectAutoFuncs

    Controls whether the injection of Func<TResult> will automatically be available without having to use the RegisterAutoFunc<TResult>(IRootTargetContainer) method or explicitly register AutoFactoryTarget targets.

    Remarks

    If this is applied to an IRootTargetContainer and the EnableAutoFuncInjection option has been configured to be true (NOTE: The default is false), then whenever a target is registered against a particular service type, a second registration will automatically be made against a Func<TResult> type with TResult equal to the registered service type.

    Also, if the EnableEnumerableInjection option is true, then an additional registration will be made for the same type for Func<TResult> where TResult is equal to IEnumerable<T> specialised for the registered type.
