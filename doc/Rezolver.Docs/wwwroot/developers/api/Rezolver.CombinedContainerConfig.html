    Class CombinedContainerConfig

    An IContainerConfig which contains zero or or more other IContainerConfig objects. Behaviours can depend on other behaviours, and this collection ensures that they are applied in the correct order.

    Remarks

    The implementation runs through each behaviour that has been added to the collection, in dependency order, calling its Configure(Container, IRootTargetContainer) method.
