    Class UnresolvedTypeCompiledTarget

    An ICompiledTarget that can be used when a type could not be resolved.

    Implementations of both GetObject(ResolveContext) and SourceTarget will throw an exception if called/read.

    Remarks

    Use of this class is encouraged when an IContainer cannot resolve a type. Instead of checking the compiled target for a null, an instance of this can be returned in its place, but its only when the GetObject(ResolveContext) method is executed that an exception occurs.

    This is particularly useful when using classes such as OverridingContainer, which allow dependencies that do not exist in the base container to be fulfilled by the overriding container instead: by delaying the throwing of exceptions until the resolve operation occurs, we are able to provide that override capability.
