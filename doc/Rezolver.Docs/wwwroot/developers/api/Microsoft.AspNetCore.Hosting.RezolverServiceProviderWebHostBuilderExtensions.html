    UseRezolver(IWebHostBuilder, Action<RezolverOptions>)

    Configures the builder so that the standard Rezolver container will be used as the DI for the web host produced by the builder, with an optional configuration callback for a RezolverOptions instance that is used to customise the behaviour of the container.

    Action<RezolverOptions> configureRezolverOptions

    Optional. Configuration callback to be applied to the RezolverOptions instance that configures the ITargetContainer and IContainer creation process.

    Note - in order for this to be applied, your application will also need to call the AddOptions(IServiceCollection) extension method (usually in your application's ConfigureServices method).
